# Chatbot-implementation-using-Chainlit

# # How to run 

1. Create a virtual env

'''bash
conda create -n llmapp python=3.11.4 -y

'''
2. Activate the conda env

'''bash
conda activate llmapp
'''

3.  Install the required packages


# #chainlit init

'''bash
pip install chainlit

chainlit init
'''

'''bash
chainlit run app.py
'''
